##### 目录树生成

tree 指令   

```bash
tree # 显示当前路径的目录，不显示子文件夹的
tree /F # 显示子文件夹的，目录树
tree /? # 查看其它参数

tree /F > xxx.txt # 将目录树生成文件，可自动新建文件

```
