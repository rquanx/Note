#### 话

互联网创业公司有一条40％规则：

年收入增长率 + 营业利润率 应该等于40％。

如果你的年增长率达到100％，那么可以承受60％的亏损。

如果年增长率为40％，你应该收支平衡。

如果增长率为20％，你应该有20％的营业利润率。

如果没有增长，你应该有40％的营业利润率。

如果业务下降10％，你应该有50％的营业利润率。

如果你快速增长，就可以接受赔钱。随着增长放缓，你必须赚钱并增加利润。
